# ADR-0002: HTML First

## Status
superseded by 0004-Adopt-Bootstrap

## Context
Currently, I am having to make a decision about what type of framework I would like to use for this project. I have very little experience in web development, and would like some more learning before I move into something like REACT, or use something like Wix where I have no idea what is going on under the hood. Although, I have a small bit of experience with things like HTML, CSS, and JS.

## Decision
We chose to just make a simple website out of HTML, CSS and JS.

## Alternatives Considered
We could have used something like a React framework, but for the time and my knowledge since this is supposed to be for my benefit, was not useful.

Or a template site builder, like Wix, but not as adaptable as I would like it to be, and also heavily constrained.

## Consequences
The learning process will be easier, and since I want to avoid black boxes in my system, and reach a full understanding of what my system is made out of.

The downside is that the creation process will be repetitive, and fairly arduous compared to using something like REACT or a Wix template.
